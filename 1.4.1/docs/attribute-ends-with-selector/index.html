Attribute Ends With Selector [name$=value]

Categories: Selectors > Attribute

attributeEndsWith selector

version added: 1.0jQuery('[attribute$=value]')

  • attribute
    An attribute name.
    value
    An attribute value. Quotes are optional.

Description: Selects elements that have the specified attribute with a value ending exactly with a given string.

Example:

Finds all inputs with an attribute name that ends with 'letter' and puts text in them.

<!DOCTYPE html>
<html>
<head>
  <script src="/scripts/jquery-1.4.js"></script>
</head>
<body>
	<input name="newsletter" />

  <input name="milkman" />
  <input name="jobletter" />
<script>$("input[name$='letter']").val("a letter");</script>
</body>
</html>
