callbacks.locked()

Categories: Callbacks Object

callbacks.locked() Returns: Boolean

Description: Determine if the callbacks list has been locked.

  • version added: 1.7callbacks.locked()

Example

Using callbacks.locked() to determine the lock-state of a callback list:

// a sample logging function to be added to a callbacks list
var foo = function( value ){
    console.log( 'foo:' + value);
}

var callbacks = $.Callbacks();

// add the logging function to the callback list
callbacks.add( foo );

// fire the items on the list, passing an argument
callbacks.fire( 'hello' );
// outputs 'foo: hello'

// lock the callbacks list
callbacks.lock();

// test the lock-state of the list
console.log ( callbacks.locked() ); //true
